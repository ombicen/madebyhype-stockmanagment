Advanced WooCommerce stock management plugin with version control, bulk editing, and comprehensive reporting.

== Description ==

Advanced WooCommerce stock management plugin with version control, bulk editing, and comprehensive reporting. Features include real-time stock tracking, price management, variation support, and detailed sales analytics with customizable date filters.

= Features =

* Real-time stock tracking
* Bulk stock editing
* Version control for stock changes
* Price management
* Variation support
* Detailed sales analytics
* Customizable date filters
* Comprehensive reporting

= Requirements =

* WordPress 5.0 or higher
* WooCommerce 6.0 or higher
* PHP 7.4 or higher

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/madebyhype-stockmanagment` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->MadeByHype Stock Management screen to configure the plugin

== Frequently Asked Questions ==

= Does this plugin work with WooCommerce? =

Yes, this plugin is specifically designed for WooCommerce and requires WooCommerce 6.0 or higher.

= Can I bulk edit stock levels? =

Yes, the plugin includes comprehensive bulk editing capabilities for stock management.

= Does it support product variations? =

Yes, the plugin fully supports WooCommerce product variations.
